HTML Cheatsheet


Common HTML elements

Tag Name Description
<abbr> Abbreviation acronym or abbreviation of a longer word or phrase, full text displayed when hovered over
<acronym> Acronym text that is displayed when hovered over (considered deprecated and no longer in use; use abbr instead)
<audio> Audio embed audio on the page
<blockquote> Blockquote section that is quoted from another source. (usually indented)
<br> Break break between lines
<button> Button
<canvas> Canvas creates graphics & animations in the page for JavasScript & webGL to interact with
<cite> Citation
<code> Code source code contained in the text
<col> Column creates properties for table columns, often used within a <colgroup>
<datalist> Datalist displays pre-defined values to a user while typing into an input box
<em> Emphasized text is converted to italics
<embed> Embed inserts external context
<form> Form interface to collect & submit user supplied info
<hr> Horizontal Rule horizontal line break between text elements
<iframe> iFrame a container used to embed a second webpage inside the current one
<img> Image
<input> Input creates an interactive element, used within a form to allow user input. Can be used to make textboxes, color pickers, date pickers, & other UI elements.
<kbd> Keyboard emphasizes characters to resemble keys on a keyboard
<label> Label Identifies captions for other elements of an HTML doc
<link> Link Connects the current page with an external file
<menu> Menu represents an unordered list
<noscript> Noscript displays content within if JavaScript is disabled or not supported
<object> Object represents external resource to be handled by a browser plugin
<ol> Ordered List
<output> Output displays the results of a calculation or user action
<p> Paragraph
<param> Parameter used to pass parameters to a resource defined in an object tag
<picture> Picture represents multiple possible img src's to be applied to different devices and screen sizes
<q> Quotation brief in-line quotation
<s> Strikethrough
<select> Select creates a drop-down list
<source> Source interface for adding source content to the page
<span> Span
<strong> Strong makes the text bold
<style> Style applies CSS styles to an HTML doc
<textarea> TextArea multi-line plaintext input
<track> Track HTML element that specifies subtitles, closed captioning and other text files for media elements
<u> Underline
<ul> Unordered List
<video> Video usually includes a source. Adding 'controls' at the end adds standard video controls. Can also include width and height. Usually includes a src.

HTML Tables

Tag Name Description
<table> Table
<tr> Table Row
<td> Table Data can span across more than one row or column using rowspan and colspan respectively. Example: <td colspan="2">Data</th> data spans across two columns.
<th> Table Heading scope attribute can take one of two values: row or col. example: <th scope="col">Heading</th>
<tbody> Table Body Contains all the table's data excluding table headings
<tfoot> Table Footer footer is often used to contain sums, differences, and other data results
